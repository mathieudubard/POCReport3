**SQL QUERIES (USE THESE IF VIEWS FAIL):**

Query 1 - Basic Reserves and Exposure (CURRENT PERIOD):
```sql

  SUM(onBalanceSheetReserve + offBalanceSheetReserve) as totalReserves,
  SUM(amortizedCost) as totalExposure,
  SUM(onBalanceSheetReserve) as onBalanceSheetReserve,
  SUM(offBalanceSheetReserve) as offBalanceSheetReserve
FROM instrumentResult 
WHERE analysisIdentifier = '[CURRENT_ANALYSIS_ID]' AND scenarioIdentifier = 'Summary'
```

Query 2 - Basic Reserves and Exposure (PRIOR PERIOD):

Query 6 - Collectively vs Individually Evaluated (CURRENT PERIOD) - **CRITICAL FOR REPORT**:
```sql
SELECT 
  ref.ascImpairmentEvaluation,
  COUNT(*) as instrument_count,
  SUM(res.amortizedCost) as totalExposure,
  SUM(res.onBalanceSheetReserveUnadjusted) as quantitativeReserve,
  SUM(res.onBalanceSheetReserveAdjusted - res.onBalanceSheetReserveUnadjusted) as qualitativeReserve,
  SUM(res.onBalanceSheetReserveAdjusted) as totalOnBalanceReserve
FROM instrumentResult res
JOIN instrumentReference ref ON res.instrumentIdentifier = ref.instrumentIdentifier
WHERE res.analysisIdentifier = '[CURRENT_ANALYSIS_ID]'
AND res.scenarioIdentifier = 'Summary'
AND res.adjustmentIdentifier = '0'
GROUP BY ref.ascImpairmentEvaluation
```

Query 7 - Collectively vs Individually Evaluated (PRIOR PERIOD) - **CRITICAL FOR REPORT**:
